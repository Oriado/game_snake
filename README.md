# game_snake

## Опис проекту
Це класична гра "Змійка", реалізована на Python з використанням бібліотеки Pygame. Гравець керує змійкою, яка повинна з'їдати яблука, щоб рости. Мета гри - набрати якомога більше очок, не врізавшись у стіни або саму себе.

## Особливості гри
- Красива графіка з неоновими кольорами яблук
- Анімовані очі змійки, що дивляться в напрямку руху
- Система підрахунку очок
- Головне меню
- Екран Game Over з можливістю перезапуску
- Два варіанти керування: стрілки та WASD

## Вимоги до системи
- Python 3.7 або новіша версія
- Pygame бібліотека

## Встановлення

### 1. Клонування репозиторію
```bash
git clone <посилання на репозиторій>
cd <game_snake>
```

### 2. Встановлення залежностей
Для запуску гри потрібно встановити бібліотеку Pygame:

```bash
pip install pygame
```

або

```bash
pip install -r requirements.txt
```

## Запуск гри
Після встановлення всіх залежностей, запустіть гру командою:

```bash
python snake_game.py
```

## Керування
- **Стрілки** ↑ ↓ ← → - керування змійкою
- **WASD** - альтернативне керування
  - W - вгору
  - S - вниз
  - A - вліво
  - D - вправо

## Правила гри
1. Змійка постійно рухається в обраному напрямку
2. З'їдайте яблука, щоб збільшити довжину змійки та отримати очки
3. За кожне яблуко ви отримуєте +10 очок
4. Гра закінчується, якщо:
   - Змійка врізається в стіну
   - Змійка врізається сама в себе

## Перезапуск гри
Після завершення гри на екрані Game Over:
- Натисніть **R** - перезапустити гру
- Натисніть **Q** - вийти з гри
